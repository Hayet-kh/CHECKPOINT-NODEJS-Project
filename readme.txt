# Node.js Exercises

This repository contains a series of Node.js exercises aimed at learning and practicing various Node.js features. Each task is implemented in its own file and demonstrates different aspects of Node.js development.

## Tasks

### 1. Hello World

- **File**: `hello-world.js`
- **Description**: A simple Node.js program that prints "HELLO WORLD" to the console.
- **Usage**: Run the program using `node hello-world.js`.

### 2. Simple HTTP Server

- **File**: `server.js`
- **Description**: Creates an HTTP server that listens on port 3000 and responds with `<h1>Hello Node!!!!</h1>\n` when accessed.
- **Usage**: Run the server using `node server.js` and visit `http://localhost:3000` in your web browser.

### 3. File System Operations

- **File**: `file-system.js`
- **Description**: Demonstrates basic file system operations in Node.js. 
  - Creates a file named `welcome.txt` with the content "Hello Node".
  - Reads the content from `hello.txt` and logs it to the console.
- **Usage**: Run the script using `node file-system.js`.

### 4. Password Generator

- **File**: `password-generator.js`
- **Description**: Generates a random password using the `generate-password` npm package and logs it to the console.
- **Installation**: Run `npm install generate-password`.
- **Usage**: Run the program using `node password-generator.js`.

### 5. Email Sender

- **File**: `email-sender.js`
- **Description**: Uses the `nodemailer` package to send an email. 
- **Installation**: Run `npm install nodemailer`.
- **Usage**: Modify the script with your email credentials and run it using `node email-sender.js`. Make sure to remove or anonymize personal information before sharing the code.
